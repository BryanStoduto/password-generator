# 🔐 Password Generator

A simple password generator built with **HTML, CSS, and JavaScript** that creates secure random passwords and allows copying them to the clipboard.

## 🚀 Features

* Generate random secure passwords
* Copy password to clipboard with one click
* Show notification when password is copied
* 16-character strong passwords

## 🛠️ Technologies

* HTML5
* CSS3
* JavaScript (ES6)
* Clipboard API

## 📌 How it works

When the user clicks the generate button, the system creates a random password using letters, numbers, and special characters. The generated password is displayed in the input field.

When the copy icon is clicked, the password is copied to the clipboard and a temporary alert message is shown confirming the action.

## ▶️ How to run

1. Clone this repository.
2. Open the `index.html` file in your browser.
3. Click the generate button to create a password.
4. Click the copy icon to copy it.
