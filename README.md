# 🔐 Gerador de Senhas

Um gerador de senhas simples desenvolvido com **HTML, CSS e JavaScript**, que cria senhas aleatórias seguras e permite copiar para a área de transferência.

## 🚀 Funcionalidades

* Gerar senhas aleatórias e seguras
* Copiar senha com um clique
* Exibir notificação ao copiar a senha
* Senhas fortes com 16 caracteres

## 🛠️ Tecnologias

* HTML5
* CSS3
* JavaScript (ES6)
* Clipboard API

## 📌 Como funciona

Ao clicar no botão de gerar, o sistema cria uma senha aleatória usando letras, números e caracteres especiais. A senha gerada é exibida no campo de input.

Ao clicar no ícone de copiar, a senha é copiada para a área de transferência e uma mensagem temporária é exibida confirmando a ação.

## ▶️ Como executar

1. Clone este repositório.
2. Abra o arquivo `index.html` no navegador.
3. Clique no botão para gerar uma senha.
4. Clique no ícone para copiar a senha.
